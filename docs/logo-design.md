# SynapseGov Logo & Branding

## Logo Concept

**Name**: SynapseGov
**Tagline**: "Instant governance connections"

**Design Concept**:
- **Synapse**: Neural connection point - represents instant connections and real-time communication
- **Visual Elements**: 
  - Neural network nodes connected by lines (representing real-time data streams)
  - Modern, tech-forward aesthetic
  - Clean, professional appearance
  - Works well in dark mode

**Color Palette**:
- Primary: Purple (#8b5cf6) - SynapseGov brand color
- Secondary: Cyan (#06b6d4) - Accent/highlights
- Dark Mode Background: #0f0f0f
- Light Mode Background: #ffffff

## Logo Variations

1. **Full Logo**: SynapseGov text + icon
2. **Icon Only**: For favicon and small spaces
3. **Dark Mode**: Optimized for dark backgrounds
4. **Light Mode**: Optimized for light backgrounds

## Favicon

- Simplified icon version
- 32x32px minimum
- Works at small sizes
- Recognizable neural network pattern
